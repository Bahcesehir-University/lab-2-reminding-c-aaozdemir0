# CMP1002 – C++ Review Lab

## Overview

This lab is designed as a short refresher on basic C++ programming concepts. You will complete several small tasks inside the provided `MainProgram.cpp` file.

The goal is to remind you of core programming ideas before we move into more advanced topics such as Object-Oriented Programming, memory management, and data structures.


---

## Learning Objectives

By completing this lab you will review:

- Basic C++ syntax
- Functions
- Loops
- Vectors
- String manipulation
- Simple class creation

## Lab Structure

The program is divided into five parts:

### Part 1 – Functions

Implement a function that returns the sum of two integers.

Skills practiced:

- Function definition
- Return values

---

### Part 2 – Loops

Write a function that calculates the sum of numbers from `1` to `n`.

Example:

```
Input: n = 5
Output: 15
```

Skills practiced:

- `for` or `while` loops
- Accumulator variables

---

### Part 3 – Vectors

Implement a function that finds the largest number in a vector.

Example:

```
Vector: {3,7,2,9,5}
Output: 9
```

Skills practiced:

- Iterating through vectors
- Comparison logic

---

### Part 4 – Strings

Write a function that reverses a string.

Example:

```
Input: "hello"
Output: "olleh"
```

Skills practiced:

- String indexing
- Loops

---

### Part 5 – Classes

Implement a simple `Student` class.

Requirements:

- Store a student's name and grade
- Implement a constructor
- Implement a function `printInfo()` that prints:

```
Name: Alice
Grade: 90
```

Skills practiced:

- Class syntax
- Constructors
- Member functions
